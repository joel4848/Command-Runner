# Command Runner

Command Runner lets you write (with command suggestions/autocomplete!) or paste in lists of commands to run in bulk, save and load presets, and schedule saved presets to run when various events trigger.

![Main Screen](https://raw.githubusercontent.com/joel4848/Command-Runner/refs/heads/main/Screenshots/Main.png)

## Features

- Write or paste in lists of commands and run them all with one simple click
- Save command lists as a preset, then load and run them with _two_ simple clicks, or one simple command!
- Schedule presets to run on various events

## Commands
- `/commandrunner` - opens the main screen
- `/commandrunner run <preset name>` - runs the specified preset
- `/commandrunner open <preset name>` - opens the main screen with the specified preset loaded
- `/commandrunner schedule <preset name>` - opens the schedule screen for the specified preset
- `/commandrunner delete <preset name` - deletes the specified preset

## Schedule events
- Every X ticks/seconds/minutes/hours
- When the game loads
- When you die
- When you respawn
- When you take damage
- When <key> is pressed
- When your health drops below x%
- When you sleep
- When you wake up
- When the weather changes

If you can think of any more, raise an issue on github and I'll add it when I see it!
